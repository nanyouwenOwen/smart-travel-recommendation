# Smart Travel Assistant v0.1.0

候选包含完整行程规划、AI 咨询和实时数据演示链路。默认 AI/实时供应商为确定性 Stub，无需密钥；尚未用真实 DeepSeek/Xiaomi MiMo 凭据验收。公共实时端点无生产 SLA；交通路线、预订、票价和签证不在 MVP 范围。Codespaces 已配置，但全新 Codespace 的远程人工验收仍未执行。部署、备份、排障和验证方法见相邻文档。

运行与部署方法见 `README.md` 和 `docs/deployment.md`。下载附件后先执行 `sha256sum -c SHA256SUMS`，并确认 `GIT_SHA` 与 Release tag 指向的提交一致。
